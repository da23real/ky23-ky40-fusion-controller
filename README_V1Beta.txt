Gamepad Windows Translator V4.2

Installation :
1) Lancer INSTALL_DEPENDANCES_V4_2.bat
2) Lancer RUN_GAMEPAD_TRANSLATOR_V4_2.bat

Mapping par défaut :
- Joy gauche = Shift + clic molette + souris
- Croix = clic molette + souris sans Shift
- Joy droit = souris normale
- LT/RT = zoom molette
- X maintenu = clic gauche maintenu
- B maintenu = clic droit maintenu
- Clic joy gauche = alterne vitesse normal/rapide du joy gauche
- Clic joy droit = alterne vitesse normal/rapide du joy droit

Nouveautés V4.2 :
- Interface par onglets
- Clamp écran activable
- Paramètres par défaut repris de la capture utilisateur
- LT/RT visibles dans leur onglet
- Debug séparé dans Général / Debug
